🎨 Bela Arte — Página de Links

Este repositório contém o código da página de links oficial da Bela Arte, criada para centralizar o acesso rápido às redes sociais, contatos e principais canais da empresa.

A página funciona como um hub de links, permitindo que clientes encontrem facilmente informações importantes em um único lugar.

📌 Sobre o projeto

A página foi desenvolvida para:

Reunir todos os links importantes da Bela Arte

Facilitar o acesso de clientes às redes sociais

Servir como link principal para bio do Instagram, WhatsApp e outros canais

Oferecer uma interface simples, rápida e responsiva

🚀 Tecnologias utilizadas

Este projeto pode utilizar tecnologias simples de frontend:

HTML5

CSS3

📱 Funcionalidades

Lista de links personalizados

Botão para WhatsApp

Links para redes sociais

Layout responsivo para celular
